# Builder vocabulary topics

Prompthaus groups terms into nine catalog topics. Use these labels when upgrading human language.

| Topic | Outcome |
| --- | --- |
| Foundations | Name core patterns |
| Prompting | Prompt clearly |
| UI Kits | Name UI tools |
| Animation | Describe motion |
| Using AI | Use AI well |
| Design | Read UI work |
| Launch | Ship confidently |
| Product | Talk product |
| Automotive | Follow car tech |

## Learners

- **CTE:** Foundations, Design, Prompting
- **CS undergrad:** Foundations, Launch, Using AI, Product
- **Working engineers:** Full catalog in the iOS app

Fetch definitions per term from `https://prompthausapp.com/api/lingo/term?name={term}`.
