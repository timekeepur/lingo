# Lingo

Agent skill for upgrading vague human prompts to precise builder vocabulary.

Prompthaus ships the full dictionary in the iOS app. This repo ships behavior only.

## Install

```bash
npx skills add timekeepur/lingo
```

Works with Cursor, Claude Code, Codex, and more.

### Cursor

If the skill does not appear after install:

```bash
npx skills add timekeepur/lingo -a cursor -g -y
```
